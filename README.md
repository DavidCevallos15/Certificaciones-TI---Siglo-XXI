## Despliegue en Vercel

Opciones para desplegar rápidamente en Vercel:

- Desde GitHub: en Vercel crea un nuevo proyecto e importa este repositorio (`DavidCevallos15/Certificaciones-TI---Siglo-XXI`). Selecciona la rama `main` y *Framework* -> `Other` (sitio estático). El *Root Directory* déjalo en `/`.
- Con la Vercel CLI (local): instala `vercel` y ejecuta desde la carpeta del proyecto:

```powershell
npm i -g vercel
cd "c:\Users\jimda\OneDrive\Documents\Web Proyectos"
vercel
```

Archivos incluidos para facilitar el despliegue:

- `index.html` — copia de la página principal lista para servir.
- `vercel.json` — configuración para servir `index.html` como sitio estático.
- `package.json` — metadatos mínimos.
